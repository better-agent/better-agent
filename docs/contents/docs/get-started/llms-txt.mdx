Better Agent publishes plain-text docs files for coding agents and other LLM tools.

## Get the Files

- `/llms.txt`
- `/llms-full.txt`

## Which One to Use

- `llms.txt` is the shorter index.
- `llms-full.txt` includes the full exported docs text.

Use the one that fits your tool and context window.
