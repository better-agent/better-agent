# Better Agent Starter

Generated by Better Agent.

Docs: https://www.better-agent.com/docs

## Included

- `src/routes/index.tsx`: starter page
- `src/routes/agents/$.tsx`: Better Agent route mounted at `/agents`
- `src/better-agent/server.ts`: Better Agent app definition
- `src/better-agent/client.ts`: typed client for the generated app

## Run it

1. Add your provider keys to `.env`.
2. Install dependencies with your package manager.
3. Start the app with `npm run dev`.
