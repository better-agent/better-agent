# Better Agent Starter

Generated by Better Agent.

Docs: https://www.better-agent.com/docs

## Included

- `app/routes/home.tsx`: starter page
- `app/routes/agents.$.tsx`: Better Agent route mounted at `/agents`
- `app/better-agent/server.ts`: Better Agent app definition
- `app/better-agent/client.ts`: typed client for the generated app

## Run it

1. Add your provider keys to `.env`.
2. Install dependencies with your package manager.
3. Start the app with `npm run dev`.
