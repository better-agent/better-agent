# Better Agent Starter

Generated by Better Agent.

Docs: https://www.better-agent.com/docs

## Included

- `app/pages/index.vue`: starter page
- `server/api/agents/[...path].ts`: Better Agent route mounted at `/api/agents`
- `lib/better-agent/server.ts`: Better Agent app definition
- `lib/better-agent/client.ts`: typed client for the generated app

## Run it

1. Add your provider keys to `.env`.
2. Install dependencies with your package manager.
3. Start the app with `npm run dev`.
